# BASH

## Built-in Variables

| Varible     | Description                                                                                    |
|-------------|------------------------------------------------------------------------------------------------|
| `$0`        | Name of script                                                                                 |
| `$1` - `$9` | Arguments to the script                                                                        |
| `$#`        | Number of arguments to script.  Does not count script name i.e. `foo.sh one two` has `$#` == 2 |
